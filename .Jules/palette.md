## 2025-05-15 - Accessible Attendance Grid
**Learning:** For interactive grid controls, use 'brightness-125' on hover and explicit focus rings (e.g., 'focus:outline-none focus:ring-2') to provide distinct visual feedback in the dark theme. Localization of status descriptions (e.g., 'keldi' for present) is crucial for consistency in this app's Uzbek interface.
**Action:** Always include descriptive ARIA labels and title tooltips for icon-only grid buttons, incorporating row context (student name) and column context (date).
